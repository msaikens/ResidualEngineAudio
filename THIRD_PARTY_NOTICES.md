Project: Opus Interactive Audio Codec  
Upstream: Xiph.Org Foundation / Opus codec project  
Website: https://opus-codec.org/  
Source used by this project: fetched by CMake from `https://github.com/xiph/opus.git`

The Opus project describes Opus as an open, royalty-free audio codec for interactive speech and audio.
